Geom3.Geom3 ( List< List< Vec3 >>  points)
inline

Construct a new 3D geometry from a list of points.

The list of points should contain sub-arrays, each defining a single polygon of points. In addition, the points should follow the right-hand rule for rotation in order to define an external facing polygon.
